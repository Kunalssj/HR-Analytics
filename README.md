# HR-Analytics
# 📊 HR Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing employee data to uncover insights related to attrition, workforce distribution, and key HR metrics. An interactive dashboard was developed using Power BI to help stakeholders make data-driven decisions and improve employee retention.

---

## 🎯 Objective
- Analyze employee data to identify attrition trends  
- Track key HR metrics such as headcount, retention, and employee distribution  
- Build an interactive dashboard for better visualization and decision-making  

---

## 🛠️ Tools & Technologies
- Power BI  
- SQL (for data analysis)  
- Microsoft Excel  
- Power Query (ETL)  
- DAX (Data Analysis Expressions)  

---

## 📂 Dataset
- Dataset includes **1000+ employee records**  
- Features: Age, Salary, Department, Job Role, Tenure, Attrition Status  

---

## ⚙️ Steps Performed
1. Data cleaning and transformation using **Power Query**  
2. Data modeling and relationship building in Power BI  
3. Creation of KPIs using **DAX** (Attrition Rate, Average Salary, etc.)  
4. Dashboard development with interactive visuals and filters  

---

## 📈 Key Insights
- Identified **15–20% higher attrition** in specific age and salary groups  
- Certain job roles showed consistently higher turnover rates  
- Mid-tenure employees had higher probability of leaving  

---

## 📊 Dashboard Features
- Attrition Analysis (by department, age, salary, job role)  
- Employee Distribution Overview  
- Key KPIs (Attrition Rate, Headcount, Avg Salary)  
- Interactive filters and slicers for dynamic analysis  

---

## 🚀 Business Impact
- Enabled data-driven HR decisions  
- Improved understanding of employee retention challenges  
- Reduced manual reporting effort through automation
